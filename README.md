# Student Management System

A simple and efficient Student Management System built using Python. This application allows users to manage student records by adding, viewing, searching, and deleting student information. Student data is stored in a JSON file, ensuring that records are preserved even after the application is closed.

## Features
- Add new student records
- View all student records
- Search students by Roll Number
- Delete student records
- Automatic data storage using JSON
- User-friendly command-line interface

## Technologies Used
- Python
- JSON
- File Handling

## Learning Outcomes
- Python fundamentals
- Functions
- Lists and Dictionaries
- JSON file handling
- CRUD Operations (Create, Read, Update, Delete)
- User input validation
- Program logic and data management

## How to Run

```bash
python student_management.py
```
